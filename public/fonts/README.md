# Fuentes Satoshi

Para usar las fuentes Satoshi en este proyecto, coloca los siguientes archivos en esta carpeta:

- `Satoshi-Regular.woff2` - Satoshi Regular (400)
- `Satoshi-Medium.woff2` - Satoshi Medium (500)

Si no tienes los archivos de Satoshi, el proyecto usará Inter como fallback automáticamente.

## Dónde obtener Satoshi

Satoshi es una fuente premium disponible en:
- [Fontshare](https://www.fontshare.com/fonts/satoshi)
- Otras distribuidoras de fuentes premium

## Formato recomendado

Se recomienda usar archivos `.woff2` para mejor rendimiento y compatibilidad.
